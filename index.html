PHILIPPINE FOOD PRICE ANALYTICS PIPELINE

Cloud ELT pipeline built on the Medallion Architecture — ingesting food price data from three real sources (WFP market CSVs, PSA CPI Excel files, and a live BSP exchange rate API) into Snowflake, transforming through dbt-modelled Bronze → Silver → Gold layers with 20 data quality tests, and scheduled monthly via a 4-task Apache Airflow DAG on Astronomer Cloud. The Gold star schema feeds a live Tableau dashboard and two Python notebooks for price trend analysis and forecasting.

Pipeline design
Three ingestion scripts load from different source formats into Snowflake Bronze as-is — WFP CSV (121,512 rows), PSA CPI multi-sheet Excel (179,289 rows), and a live BSP API call that appends one new exchange rate row per monthly run rather than overwriting. dbt handles all transformations: Silver models standardize column types, deduplicate via ROW_NUMBER(), and unpivot the PSA CPI from wide to long format. Gold builds a star schema — dim_region, dim_commodity, dim_date, dim_market + fact_food_prices — with surrogate keys and a price-to-CPI ratio column as the core analytical metric. The 4-task Airflow DAG (ingest → silver → gold → dbt test) runs on the 1st of every month; if any dbt test fails, the pipeline halts before bad data reaches Gold.
Data quality
20 dbt tests cover not_null, unique, accepted_values, and referential integrity across all Silver and Gold models. Tests run as the final DAG task — PASS=20, WARN=0, ERROR=0. The Bronze layer is never overwritten, preserving a full raw audit trail for every monthly run.
Analytics & findings
BSP lag correlation analysis: peso depreciation leads food price increases by 1–2 months (r=0.29, p=0.018). Prophet forecasting with and without BSP as regressor diverges during depreciation periods — confirming exchange rate adds genuine forecast signal. Key findings: NCR food prices run 60+ index points above PSA CPI. Meat, fish, and eggs surged 80.6% from 2018–2023. Luzon is 19.7% more expensive than Mindanao for the same WFP basket.

EUROPEAN FASHION RETAIL DATA WAREHOUSE

End-to-end SQL data warehouse built entirely in MySQL and Python — no cloud services, no orchestration tools. A single python main.py command runs the full 4-stage pipeline (Bronze → Silver → Gold) wrapped in a MySQL transaction that rolls back on any failure. Stored procedures handle all Silver transformations. The Gold star schema is built with foreign key constraints, indexed fact table, and 4 BI-ready views, supporting 14 business analytics queries across revenue, products, customers, campaigns, and inventory.

Pipeline design
Bronze: 7 tables loaded via LOAD DATA LOCAL INFILE (fastest MySQL bulk-load method), truncated and reloaded on each run for idempotency. Silver: a stored procedure (transform_silver_layer()) applies type casting for all IDs, dates, and decimals; strips percentage characters; trims whitespace; and removes duplicates via ROW_NUMBER() — all in one transactional call. Gold: star schema with dim_customer, dim_product, dim_campaign + fact_sales (2,253 rows), surrogate keys, foreign keys, and indexed on sale_date and customer_id. Campaign attribution joins on channel + date range so each line item is automatically attributed to the active campaign — no manual tagging. Full pipeline wraps every stage in a single MySQL transaction — any failure triggers a complete rollback.
Data quality & execution
SQL-based quality tests: row count checks per layer, null checks on all primary keys, referential integrity validation, duplicate checks on dimension keys, and discount logic validation. Pipeline logs every stage to pipeline.log with timestamps — exits code 1 on failure, code 0 on success.
Business findings
€324,236 revenue · 905 orders · €358 AOV. Only 9.8% of orders were discounted (avg 24.3% off) — 92.7% of revenue is full-price, meaning healthy margin discipline. 55% of product-country inventory positions at critical stock levels, with the stockout risk query identifying which high-velocity SKUs are also most exposed — the highest-replenishment-priority items.

PSA LABOR FORCE SURVEY EDUCATIONAL MOBILITY PIPELINE & BAYESIAN ANALYSIS

Built as an undergraduate thesis — an ETL pipeline that processes 11.28 million rows of PSA Labor Force Survey microdata across 48 quarterly files (2010–2024), then applies a Bayesian Multilevel Model to measure how strongly a parent's education predicts their child's education across all 17 Philippine regions. The engineering challenge: PSA changed column names across survey redesigns, and the K-12 reform in 2016 changed what education codes mean mid-dataset — both required custom schema handling before any modelling could happen.

PythonPyMC · Bambi ArviZETL Pipeline PSA LFS microdata11.28M rows Bayesian MLM
GitHub
ETL pipeline
PSA used three different column schemas across survey years, and the K-12 reform created two incompatible grade code systems mid-2016 — the same numeric code means a different education level depending on the quarter. A schema config registry routes each file to the correct column names and grade map automatically. A 12-step cleaning pipeline then pairs parents and children within households, applies age gap filters to remove implausible pairs, and logs every run. python main.py processes all 48 files end-to-end.
Bayesian model
Multilevel Linear Model via Bambi/PyMC: 4 chains × 2,000 draws, all R-hat < 1.01. Predictors: father's and mother's schooling years (with quadratic terms), children per household, child age, and survey year fixed effects. Region-level random intercepts and random slopes allow the father-education effect to vary by region. Prior sensitivity confirmed across 3 specifications.
Key findings
For every additional year of a father's education, a son completes 0.265 more years of schooling on average — a substantial national-level persistence. Regional slopes range 0.244–0.280 but all credible intervals overlap, meaning this is a structural national problem, not isolated to any one region. Mother's education has a stronger and more regionally variable effect (0.249 in NCR to 0.385 in Davao). BARMM has the lowest average son schooling nationally at 8.86 years.
