⭐ Main project

Philippine Food Price Data Pipeline

End-to-end data engineering pipeline tracking food price trends in the Philippines. Pulls live USD/PHP rates from the BSP API, combines Kaggle food price data with PSA's official price index, loads into Snowflake, and transforms through a medallion architecture using dbt. Scheduled daily with Apache Airflow.

Python Apache Airflow Snowflake dbt REST API Kaggle datasets PSA data SQL
GitHub
Problem
Food prices in the Philippines are reported across multiple fragmented sources — no single pipeline unified them for time-series analysis or inflation tracking.
What was built
A fully automated ELT pipeline: Python scripts ingest data from 3 sources, load raw files to Snowflake's bronze layer, then dbt models transform through silver (cleaning, currency conversion) to gold (analytics-ready aggregates). Airflow runs the full DAG daily at 6 AM.
Key engineering decisions
Medallion architecture keeps raw data immutable for audit. dbt handles all SQL transforms with version-controlled models and built-in tests. Airflow provides dependency management and retry logic.
Results
Produces daily-refreshed price trend tables broken down by commodity category. Enables inflation analysis across time periods correlated with USD/PHP rate movements.

Bayesian Hierarchical Modelling — Educational Mobility PH

Undergraduate thesis. Used nationally representative labor force survey microdata to estimate intergenerational educational mobility across Philippine regions. Hierarchical Bayesian models borrow strength across small subgroups — producing credible interval estimates where classical methods break down.

Python PyMC MCMC LFS microdata Pandas ArviZ
GitHub
Problem
Estimating mobility metrics per Philippine region from survey data — small regional sample sizes make classical regression unreliable.
Approach
Processed LFS microdata, designed hierarchical priors, fitted model with MCMC via PyMC, performed posterior predictive checks and sensitivity analysis on prior choices.
Engineering note
The data pipeline for this project — ingesting, reshaping, and cleaning the LFS microdata — was itself a substantial engineering task involving design weights, multi-stage sampling adjustments, and reproducible Python scripts.
Results
Region-level mobility estimates with credible intervals. Identified areas with statistically significant deviations and policy-relevant differences across generations.
